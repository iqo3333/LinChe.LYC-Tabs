# ブラウザの互換性

このウェブサイトは、HTML5、CSS3、ES6 JavaScript、そして一部の高度なCSS機能（`will-change` など）を使用して構築されています。

機能と効果を完全に表示するためには、以下の最低サポートブラウザバージョンが必要です：

| 機能                                               | Chrome       | Edge (Chromium) | Safari         | IE           |
|--------------------------------------------------|--------------|-----------------|----------------|--------------|
| **ES6構文** (`const/let`、アロー関数、テンプレートリテラル)   | ✅ 49+        | ✅ 15+           | ✅ 10+          | ❌ サポートされていない |
| **Unicode 正規表現** (`\p{L}`, `\p{N}` ドメイン名検出用)  | ✅ 64+        | ✅ 79+           | ✅ 12+          | ❌ サポートされていない |
| **`will-change`**                                  | ✅ 36+        | ✅ 79+           | ✅ 9+           | ⚠️ 部分的なサポート |
| **Flexbox レイアウト**                              | ✅ 29+        | ✅ 12+           | ✅ 9+           | ⚠️ 基本的なサポート |
| **`MutationObserver`**                              | ✅ 11+        | ✅ 79+           | ✅ 12+          | ⚠️ 部分的なサポート |
| **背景動画（`<video>` タグ）**                     | ✅ 4+         | ✅ 12+           | ✅ 9+           | ❌ サポートされていない |
| **イベントリスナーおよび DOM 操作**                | ✅ 49+        | ✅ 15+           | ✅ 10+          | ⚠️ 部分的なサポート |

⚠️ 注意:

* IE 系列ブラウザは ES6、CSS 変数、Unicode 正規表現をサポートしておらず、ページ機能が正常に動作しない可能性があります。
* 背景動画：`<video>` タグは IE 9 以降でサポートされていますが、古い IE バージョンでは動画背景が完全にサポートされていない場合があります。
* イベントリスナーおよび DOM 操作：これらの JavaScript 機能は現代のブラウザで広くサポートされていますが、IE では `MutationObserver` などのサポートが制限されています。
* `will-change`：現代のブラウザではアニメーションや遷移の最適化に使用されますが、IE では部分的にしかサポートされておらず、パフォーマンスが限られる場合があります。
* 古い Safari (<12) および Chromium Edge (<79) は Unicode ドメイン名やキーワードを正しく認識できない場合があります。  
* 古いブラウザでもウェブサイトを開けますが、視覚効果や入力検証機能が制限されることがあります。

**Chrome 100+ / Safari 15+ / Edge 100+** を使用することをお勧めします。最適な体験とアニメーション効果を得るために。

⚠️ 注意：このレポートは「Ver.9.0.0」およびそれ以前のバージョンに適用され、以降のバージョンの互換性を保証するものではありません！
